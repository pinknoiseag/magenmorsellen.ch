## Shoot-Setup (Entwurf)

| Parameter | Vorschlag |
|-----------|-----------|
| Ort | Studio / Küche — kontrolliertes Licht |
| Hintergrund | Einheitlich (Creme oder Holz, passend zu Design-Brief) |
| Licht | Weich, von oben/seitlich — jpstern-Standard |
| Ablage Roh | `assets/raw/` (nicht in Git) |
| Export Web | `web/public/images/` via optimize-Script |

## Checkliste Shoot-Tag

- [ ] Hintergrund und Licht für alle Sorten **identisch**
- [ ] Sortenname auf Liste abhaken (12 Hauptsorten)
- [ ] Geschenksäckli alle 4 Varianten
- [ ] 1–2 Hero-Motive extra (freier komponiert)
- [ ] RAW + Auswahl-Export sichern
